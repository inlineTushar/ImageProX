Face Processing Flow

  1. Detect faces in the image using ML Kit Face Detection
  2. For each detected face, extract the bounding rectangle
  3. Crop the face region from the original image
  4. Apply a black & white (grayscale) filter to the cropped face
  5. Replace the original face region with the filtered version
  6. Save the composite result to storage
  7. Add entry to processing history with metadata

Document/PDF Scanning Flow

  1. Detect text regions using ML Kit Text Recognition
  2. Identify document edges using edge detection algorithm
  3. Apply perspective transformation to correct document angle
  4. Crop to the detected document boundaries
  5. Apply contrast enhancement for better readability
  6. Convert the processed image to PDF format
  7. Save PDF to device storage and update history
